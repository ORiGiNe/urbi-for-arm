What is overLIB

overLIB is a JavaScript library created to enhance websites with small
popup information boxes (like tooltips) to help visitors around your
website. It can be used to provide the user with information about
what will happen when they click on a link as well as navigational
help.

http://www.bosrup.com/web/overlib/

This is overLIB 4.21, released on 2005-7-05.
